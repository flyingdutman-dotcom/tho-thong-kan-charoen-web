## 🚀 เริ่มต้นอย่างรวดเร็ว

### 1. ล็อกอินเข้าระบบ

```
URL: https://[your-domain].manus.space
ปุ่ม: ล็อกอิน (มุมบนขวา)
```

### 2. ยกระดับเป็นแอดมิน

ติดต่อทีม Manus เพื่อให้ยกระดับบัญชีของคุณเป็นแอดมิน

### 3. เข้าแผงควบคุม

```
URL: https://[your-domain].manus.space/admin
```

## 📋 ฟีเจอร์หลัก

### 📊 Dashboard (/admin)
- สถิติภาพรวม (ใบขอ, ผลงาน, อัตราการตอบสนอง)
- ใบขอล่าสุด
- ปุ่มลัดเข้าถึงฟีเจอร์อื่น

### 📝 จัดการใบขอ (/admin/inquiries)
- ดูรายการใบขอเสนอราคาทั้งหมด
- ฟิลเตอร์ตามสถานะ
- อัปเดตสถานะใบขอ
- **ส่งออก CSV/Excel** ✨ ใหม่

### 🖼️ จัดการผลงาน (/admin/portfolio)
- เพิ่มผลงานใหม่
- **อัปโหลดรูปภาพ** ✨ ใหม่
- ดูรูปภาพก่อน-หลัง
- ดูรายการผลงาน

## 🔐 ความปลอดภัย

- ✅ เฉพาะแอดมินเท่านั้นที่เข้าถึงได้
- ✅ ตรวจสอบสิทธิ์ทั้ง frontend และ backend
- ✅ ข้อมูลที่ส่งออกปลอดภัย

## 🆘 ต้องการความช่วยเหลือ?

### ปัญหาทั่วไป

| ปัญหา | วิธีแก้ |
|------|--------|
| ไม่สามารถเข้า /admin | ตรวจสอบว่าเป็นแอดมิน, ล็อกออกแล้วล็อกอินใหม่ |
| อัปโหลดรูปภาพไม่ทำงาน | ตรวจสอบขนาด (≤5MB), ประเภท (JPG/PNG/WebP) |
| ส่งออกข้อมูลไม่ทำงาน | ตรวจสอบว่ามีข้อมูล, ลองฟิลเตอร์ "ทั้งหมด" |
