# ตรวจสอบการตั้งค่าแอดมิน

## ขั้นตอนการตรวจสอบ

### 1. ตรวจสอบบัญชีผู้ใช้ในฐานข้อมูล

```sql
-- ดูรายชื่อผู้ใช้ทั้งหมด
SELECT id, openId, name, email, role, createdAt FROM users;

-- ดูแอดมินเท่านั้น
SELECT id, openId, name, email, role, createdAt FROM users WHERE role = 'admin';

-- ดูข้อมูลผู้ใช้เฉพาะคน
SELECT id, openId, name, email, role, createdAt FROM users WHERE email = 'your-email@example.com';
```

### 2. ตรวจสอบการเข้าถึงแผงควบคุม

1. ล็อกอินด้วยบัญชีแอดมิน
2. ไปที่ https://[your-domain].manus.space/admin
3. ตรวจสอบว่าสามารถเห็น:
   - Dashboard (/admin)
   - จัดการใบขอเสนอราคา (/admin/inquiries)
   - จัดการผลงาน (/admin/portfolio)

### 3. ตรวจสอบฟีเจอร์แอดมิน

#### Dashboard
- [ ] แสดงสถิติจำนวนใบขอ
- [ ] แสดงจำนวนผลงาน
- [ ] แสดงใบขอล่าสุด
- [ ] ปุ่มลัดทำงาน

#### จัดการใบขอ
- [ ] แสดงรายการใบขอทั้งหมด
- [ ] ฟิลเตอร์ตามสถานะทำงาน
- [ ] สามารถอัปเดตสถานะ
- [ ] ปุ่มส่งออก CSV/Excel ทำงาน

#### จัดการผลงาน
- [ ] สามารถเพิ่มผลงานใหม่
- [ ] อัปโหลดรูปภาพทำงาน
- [ ] แสดงรูปภาพตัวอย่าง
- [ ] บันทึกผลงานลงฐานข้อมูล

### 4. ตรวจสอบการป้องกัน

1. ล็อกออก
2. ล็อกอินด้วยบัญชี "user" ทั่วไป
3. พยายามเข้า /admin
4. ตรวจสอบว่า:
   - [ ] ถูกเปลี่ยนเส้นทางไปหน้า 404 หรือหน้าแรก
   - [ ] ไม่สามารถเห็นแผงควบคุมแอดมิน

## การแก้ไขปัญหา

### ปัญหา: ไม่สามารถเห็นแผงควบคุมแอดมิน

**สาเหตุที่เป็นไปได้:**
- บัญชีไม่ได้ถูกยกระดับเป็นแอดมิน
- แคชเบราว์เซอร์ยังคงเก็บข้อมูลเก่า

**วิธีแก้:**
1. ตรวจสอบว่า role = 'admin' ในฐานข้อมูล
2. ล็อกออกแล้วล็อกอินใหม่
3. ล้างแคชเบราว์เซอร์ (Ctrl+Shift+Delete)
4. ลองใช้เบราว์เซอร์อื่น

### ปัญหา: ปุ่มอัปโหลดรูปภาพไม่ทำงาน

**สาเหตุที่เป็นไปได้:**
- ไฟล์ใหญ่เกิน 5MB
- ประเภทไฟล์ไม่รองรับ
- ปัญหาการเชื่อมต่อ S3

**วิธีแก้:**
1. ตรวจสอบขนาดไฟล์
2. ใช้รูปภาพ JPG, PNG, WebP หรือ GIF
3. ตรวจสอบการเชื่อมต่ออินเทอร์เน็ต
4. ตรวจสอบ browser console สำหรับข้อความแสดงข้อผิดพลาด

### ปัญหา: ส่งออกข้อมูลไม่ทำงาน

**สาเหตุที่เป็นไปได้:**
- ไม่มีข้อมูลใบขอในระบบ
- เบราว์เซอร์บล็อกการดาวน์โหลด

**วิธีแก้:**
1. ตรวจสอบว่ามีใบขอในระบบ
2. ลองฟิลเตอร์ "ทั้งหมด"
3. ตรวจสอบการตั้งค่าเบราว์เซอร์
4. ลองใช้เบราว์เซอร์อื่น

## ข้อมูลการเชื่อมต่อฐานข้อมูล

หากต้องการเข้าถึงฐานข้อมูลโดยตรง:

1. ไปที่ Management UI
2. คลิก "Database" panel
3. ดูข้อมูลการเชื่อมต่อในมุมล่างซ้าย
4. เปิดใช้ SSL หากจำเป็น

## การสำรองข้อมูล

**ข้อมูลที่สำคัญ:**
- ตารางผู้ใช้ (users)
- ตารางใบขอ (inquiries)
- ตารางผลงาน (portfolio)

**การสำรองอัตโนมัติ:**
- Manus จะสำรองข้อมูลโดยอัตโนมัติ
- ติดต่อ Manus Support สำหรับการสำรองด้วยตนเอง
