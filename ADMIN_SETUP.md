# คู่มือการตั้งค่าบัญชีแอดมิน - บริษัท ท่อทองการเจริญ จำกัด

## ขั้นตอนการตั้งค่าบัญชีแอดมิน

### 1. ล็อกอินเข้าระบบเป็นครั้งแรก

1. เปิดเว็บไซต์ที่ https://[your-domain].manus.space
2. คลิกปุ่ม "ติดต่อเรา" หรือ "ขอใบเสนอราคา" เพื่อไปยังหน้าแรก
3. คลิกปุ่ม "ล็อกอิน" ที่มุมบนขวา
4. ล็อกอินด้วยบัญชี Manus OAuth ของคุณ (Google, GitHub, หรือ Manus Account)

### 2. ตรวจสอบบัญชีของคุณ

หลังจากล็อกอินครั้งแรก:
- ระบบจะสร้างบัญชีผู้ใช้อัตโนมัติ
- บัญชีจะมีบทบาท "user" เป็นค่าเริ่มต้น

### 3. ยกระดับบัญชีเป็นแอดมิน

**วิธีที่ 1: ผ่านฐานข้อมูล (ผู้ดูแลระบบ Manus)**

ติดต่อทีม Manus Support เพื่อให้ยกระดับบัญชีของคุณเป็นแอดมิน:

```
ขั้นตอน:
1. ให้ทีม Manus ทราบ OpenID ของคุณ (ค่านี้ปรากฏในหน้าผู้ใช้)
2. ทีม Manus จะเรียกใช้ SQL เพื่ออัปเดต role จาก 'user' เป็น 'admin'
3. ล็อกออกแล้วล็อกอินใหม่เพื่อให้การเปลี่ยนแปลงมีผล
```

**วิธีที่ 2: ผ่านฐานข้อมูลโดยตรง (หากคุณมีสิทธิ์เข้าถึง)**

```sql
-- ค้นหาบัญชีผู้ใช้ของคุณ
SELECT id, openId, name, email, role FROM users WHERE email = 'your-email@example.com';

-- ยกระดับเป็นแอดมิน
UPDATE users SET role = 'admin' WHERE openId = 'your-openid-here';

-- ตรวจสอบการเปลี่ยนแปลง
SELECT id, openId, name, email, role FROM users WHERE openId = 'your-openid-here';
```

### 4. เข้าถึงแผงควบคุมแอดมิน

หลังจากยกระดับเป็นแอดมิน:

1. ล็อกออกจากระบบ
2. ล็อกอินใหม่
3. คลิกปุ่ม "ล็อกอิน" ที่มุมบนขวา
4. ไปที่ `/admin` หรือคลิกลิงก์ "แผงควบคุมแอดมิน" (จะปรากฏเฉพาะแอดมิน)

## ฟีเจอร์แผงควบคุมแอดมิน

### Dashboard (/admin)
- **สถิติภาพรวม**: จำนวนใบขอใหม่, ใบขอทั้งหมด, จำนวนผลงาน, อัตราการตอบสนอง
- **ใบขอล่าสุด**: แสดง 5 ใบขอล่าสุดพร้อมรายละเอียด
- **ปุ่มลัด**: เข้าถึงการจัดการใบขอและผลงานอย่างรวดเร็ว

### จัดการใบขอเสนอราคา (/admin/inquiries)
- **ดูใบขอทั้งหมด**: แสดงรายการใบขอเสนอราคาทั้งหมด
- **ฟิลเตอร์ตามสถานะ**: ใหม่, ติดต่อแล้ว, เสร็จสิ้น, ปฏิเสธ
- **อัปเดตสถานะ**: เปลี่ยนสถานะของใบขอ
- **ส่งออกข้อมูล**: ดาวน์โหลด CSV หรือ Excel ของใบขอทั้งหมด

### จัดการผลงาน (/admin/portfolio)
- **เพิ่มผลงานใหม่**: กรอกแบบฟอร์มเพื่อเพิ่มผลงาน
- **อัปโหลดรูปภาพ**: อัปโหลดรูปภาพก่อน-หลังโดยตรง
- **ดูผลงานทั้งหมด**: แสดงกริดของผลงานที่เพิ่มแล้ว
- **แก้ไขและลบ**: ปรับปรุงหรือลบผลงาน (ฟีเจอร์กำลังพัฒนา)

## วิธีใช้งาน

### การเพิ่มผลงานใหม่

1. ไปที่ `/admin/portfolio`
2. กรอกแบบฟอร์ม:
   - **ชื่อโครงการ**: ชื่อของผลงาน (เช่น "ลอกท่อระบายน้ำ ที่ บ้านเลขที่ 123")
   - **หมวดหมู่**: เลือกประเภท (ลอกท่อระบายน้ำ, ท่อประปา, บำรุงรักษา, อื่นๆ)
   - **ที่อยู่**: สถานที่ของโครงการ
   - **คำอธิบาย**: รายละเอียดเพิ่มเติม
   - **รูปภาพก่อน**: อัปโหลดรูปภาพก่อนลอก
   - **รูปภาพหลัง**: อัปโหลดรูปภาพหลังลอก
3. คลิก "เพิ่มผลงาน"

### การส่งออกข้อมูลใบขอ

1. ไปที่ `/admin/inquiries`
2. (ตัวเลือก) ฟิลเตอร์ตามสถานะ
3. คลิกปุ่ม "CSV" หรือ "Excel" เพื่อดาวน์โหลด
4. ไฟล์จะบันทึกลงคอมพิวเตอร์ของคุณพร้อมชื่อไฟล์และวันที่

## ความปลอดภัยและสิทธิ์

- **เฉพาะแอดมินเท่านั้น** ที่สามารถเข้าถึงแผงควบคุม
- **บทบาท "user"** ไม่สามารถเข้าถึง `/admin` ได้
- **การตรวจสอบสิทธิ์** ทำขึ้นทั้งที่ frontend และ backend
- **ข้อมูลที่ส่งออก** จะรวมเฉพาะข้อมูลที่ผู้ใช้ปัจจุบันได้รับอนุญาต

## การแก้ไขปัญหา

### ไม่สามารถเข้าถึง /admin
- ตรวจสอบว่าบัญชีของคุณมีบทบาท "admin"
- ลองล็อกออกแล้วล็อกอินใหม่
- ลบแคชเบราว์เซอร์และลองใหม่

### ปุ่มอัปโหลดรูปภาพไม่ทำงาน
- ตรวจสอบขนาดไฟล์ (ต้องไม่เกิน 5MB)
- ตรวจสอบประเภทไฟล์ (ต้องเป็น JPG, PNG, WebP, หรือ GIF)
- ตรวจสอบการเชื่อมต่ออินเทอร์เน็ต

### ไม่สามารถส่งออกข้อมูล
- ตรวจสอบว่ามีข้อมูลใบขอในระบบ
- ลองฟิลเตอร์ "ทั้งหมด" แทนการฟิลเตอร์เฉพาะ
- ตรวจสอบว่าเบราว์เซอร์อนุญาตให้ดาวน์โหลดไฟล์
